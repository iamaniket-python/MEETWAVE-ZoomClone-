# Zoom Clone (Video Meeting App)

A full-stack video meeting application inspired by **Zoom**, built using **React** for the frontend and **Express.js** for the backend.
This project allows users to create and join video meetings, communicate in real time, and experience basic video conferencing features.

---

## 🚀 Features

* Create a new meeting room
* Join meetings using a meeting ID
* Real-time video and audio communication
* Multiple participants in a room
* Mute / Unmute microphone
* Turn camera on / off
* Responsive UI for better user experience
* Real-time communication using WebSockets

---

## 🛠 Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Express.js
* Node.js

### Real-Time Communication

* WebRTC
* Socket.io
